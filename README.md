# Case Study: Analyzing Customer Churn in Excel

This project is part of a DataCamp case study focused on analyzing customer churn for a fictional telecom company called **Databel**. The goal is to investigate churn behaviour, identify key drivers, and visualize insights through an Excel dashboard.

## Project Overview
Subscription-based businesses rely heavily on keeping customers engaged and minimizing churn. In this project, I:
- Explored a customer dataset using Excel
- Created calculated columns and fields
- Built PivotTables to break down churn behaviour
- Designed a clean, interactive dashboard to highlight insights

## Skills Demonstrated
- Data cleaning and transformation in Excel  
- Calculated columns & DAX-like Excel logic  
- PivotTables & PivotCharts  
- Dashboard design and formatting  
- Analytical storytelling

## Project File
- **Dashboard - Customer_Churn_Rate.xlsx**

## Key Business Questions Answered
- What is the overall churn rate?
- Which customer segments churn the most?
- How do contract type, tenure, and monthly charges impact churn?
- What recommendations can reduce churn?

## About the Dataset
The dataset is provided by **DataCamp** for an Excel case study and simulates customer behaviour for a telecom company.
